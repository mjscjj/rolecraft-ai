# RoleCraft AI 零提示词体验 - 引导式创建向导使用指南

## 📖 概述

引导式创建向导通过**问答对话**代替复杂的提示词编写，让用户在 3 步之内轻松创建专业的 AI 角色。无需任何提示词工程经验，人人都能打造专属 AI 助手。

---

## 🎯 核心优势

### 传统方式 vs 引导向导

| 传统方式 | 引导向导 |
|---------|---------|
| ❌ 需要理解提示词结构 | ✅ 只需回答简单问题 |
| ❌ 手动编写系统提示词 | ✅ 自动生成专业提示词 |
| ❌ 反复调试优化 | ✅ 实时测试 + 智能推荐 |
| ❌ 学习成本高 | ✅ 零门槛上手 |

## 🚀 快速开始

### 第 1 步：基础信息

#### 1.1 AI 助手名称
- **作用**：用户看到的第一印象
- **建议**：
  - 简洁易记（2-5 个字）
  - 体现功能定位（如：营销专家、办公助手）
  - 避免生僻字和复杂英文

**示例**：
```
✅ 好：办公小助手、营销专家、健身教练
❌ 差：AI-Assistant-v2.0、超级无敌全能王
```

#### 1.2 主要用途选择
6 种预设场景，单选：

| 用途 | 适用场景 | 推荐配置 |
|-----|---------|---------|
| 📋 **智能助理** | 日常办公、日程管理、邮件撰写 | 专业严谨/友好亲切 |
| 🎯 **专业顾问** | 法律咨询、财务规划、技术指导 | 专业严谨/详细周全 |
| ✍️ **内容创作** | 文案写作、故事创作、营销内容 | 幽默风趣/激励鼓舞 |
| 📚 **教学辅导** | 知识讲解、学习辅导、技能培训 | 友好亲切/详细周全 |
| 💙 **情感陪伴** | 聊天解闷、情感支持、心理疏导 | 友好亲切/温暖耐心 |
| 📊 **数据分析** | 数据处理、报告生成、商业分析 | 专业严谨/简洁直接 |

#### 1.3 说话风格选择
6 种风格，单选：

| 风格 | 特点 | 适用场景 |
|-----|------|---------|
| 👔 **专业严谨** | 正式、准确、条理清晰 | 商务、法律、医疗 |
| 😊 **友好亲切** | 温暖、耐心、易于接近 | 客服、教育、陪伴 |
| 😄 **幽默风趣** | 轻松、有趣、富有创意 | 娱乐、创意、营销 |
| ⚡ **简洁直接** | 高效、直接、不啰嗦 | 技术、数据、查询 |
| 📝 **详细周全** | 全面、深入、注重细节 | 教学、咨询、指导 |
| 🌟 **激励鼓舞** | 积极、向上、充满能量 | 健身、职业发展 |

**进度提示**：
- ✅ 完成本步后，进度自动保存
- ✅ 可随时返回修改
- ✅ 进度指示器清晰显示当前位置

### 第 2 步：能力配置

#### 2.1 专业知识选择
**多选**，建议选择 1-3 个核心领域：

| 领域 | 能力描述 | 典型应用 |
|-----|---------|---------|
| 💼 **商务办公** | 邮件、文档、会议、项目管理 | 行政助理、项目经理 |
| 📢 **市场营销** | 策划、文案、推广、品牌 | 营销顾问、文案写手 |
| 💻 **技术编程** | 开发、调试、架构、算法 | 编程导师、技术顾问 |
| 🎨 **创意设计** | UI/UX、平面、创意构思 | 设计师、创意总监 |
| 💰 **财务金融** | 会计、投资、理财、税务 | 财务顾问、投资分析师 |
| ⚖️ **法律法务** | 合同、合规、法律咨询 | 法务顾问、律师助理 |
| 👥 **人力资源** | 招聘、培训、绩效、员工关系 | HR 顾问、招聘专员 |
| 🏥 **健康医疗** | 健身、营养、心理健康 | 健身教练、健康顾问 |
| 🎓 **教育培训** | 课程、辅导、学习方法 | 教师、教育顾问 |
| 🌴 **生活休闲** | 旅行、美食、购物、娱乐 | 旅行规划师、生活达人 |

**⚠️ 重要提醒**：
- 选择「法律法务」「健康医疗」「财务金融」时，系统会自动添加免责声明
- 专业领域过多（>5 个）可能导致角色定位模糊

#### 2.2 应避免事项选择
**多选**，帮助 AI 避免常见错误：

| 避免事项 | 说明 | 推荐场景 |
|---------|------|---------|
| 🎯 **猜测臆断** | 不确定的信息要明确说明 | 专业咨询、医疗健康 |
| 🔄 **重复啰嗦** | 避免重复相同内容 | 商务沟通、技术支持 |
| 📖 **专业术语** | 少用晦涩难懂的专业词汇 | 大众科普、新手教学 |
| ⚠️ **敏感话题** | 避开政治、宗教等敏感议题 | 客服、公共场景 |
| 🚫 **过度承诺** | 不夸大能力，诚实告知局限 | 所有场景（强烈推荐） |
| ⚖️ **主观偏见** | 保持客观中立，不带个人偏见 | 咨询、分析、评估 |

#### 2.3 特殊要求（可选）
输入框，支持自由文本：

**示例**：
```
• 每次回答前先确认理解了我的问题
• 使用 Markdown 格式输出代码，并添加详细注释
• 涉及数据时优先使用图表展示
• 回答控制在 300 字以内，简洁明了
• 遇到复杂问题时分步骤说明
```

#### 2.4 智能推荐提示
系统根据已选配置，实时显示推荐：

**推荐类型**：
- 💡 **配置建议**：基于用途和风格推荐技能
- ⚠️ **风险提醒**：特定领域的注意事项
- ✨ **最佳实践**：行业通用优化方案

**示例**：
```
💡 AI 智能推荐：
• 建议开启「日程管理」和「邮件撰写」技能
• 说话风格推荐「专业严谨」或「友好亲切」
• 重要：添加免责声明「不构成正式法律意见」
```

### 第 3 步：测试优化

#### 3.1 实时测试对话框
模拟真实对话场景：

**操作步骤**：
1. 在输入框输入测试消息
2. 点击「发送」或按 Enter
3. 查看 AI 助手的回复
4. 进行满意度评分

**测试建议**：
```
✅ 好问题：
• "帮我写一封会议邀请邮件"
• "如何提升产品转化率？"
• "这份合同有什么风险？"

❌ 避免：
• "你好"（太简单）
• 超长复杂问题（难以评估）
```

#### 3.2 满意度评分
- 👍 **满意**：记录正面反馈，强化当前配置
- 👎 **不满意**：触发优化建议

#### 3.3 三种完成路径

| 选项 | 操作 | 适用场景 |
|-----|------|---------|
| ✅ **一键完成** | 直接创建角色 | 对测试结果满意 |
| 🔄 **返回调整** | 返回上一步修改 | 需要微调配置 |
| ✨ **AI 优化** | 获取智能优化建议 | 不确定如何改进 |

**AI 优化功能**：
- 分析当前配置的优缺点
- 提供具体改进建议
- 展示类似成功案例

## 🔧 后端支持

### 1. 问答转提示词算法

**输入**：用户的选择和回答
**处理**：
1. 结构化数据映射
2. 模板匹配和填充
3. 风格参数调整
4. 免责声明注入
**输出**：完整的系统提示词

**生成示例**：
```markdown
# 角色设定：营销专家

## 核心定位
你是一位专业的营销策划助手。你的主要职责是帮助用户制定营销策略、撰写文案。

## 说话风格
幽默风趣。在交流中，你应该轻松、有趣、富有创意。

## 专业领域
你擅长以下领域：市场营销。在这些领域内，你应该提供专业、准确的建议和信息。

## 应避免事项
请注意避免以下情况：
- **过度承诺**: 不夸大能力，诚实告知局限
- **猜测臆断**: 不确定的信息要明确说明

## 行为准则
1. 始终以帮助用户为首要目标
2. 如遇不确定的信息，诚实告知而非猜测
3. 保持专业且友好的态度
4. 回答应清晰、有条理、实用

## 开始
现在，请以营销专家的身份，用幽默风趣的方式，开始为用户提供帮助。
```

### 2. 智能推荐逻辑

**推荐引擎**：
- **基于规则**：预设的专家知识库
- **基于场景**：用途 + 风格 + 领域的组合匹配
- **基于风险**：特定领域的合规要求

**推荐优先级**：
- 🔴 **High**：必须注意（如法律免责声明）
- 🟡 **Medium**：建议优化（如技能配置）
- 🟢 **Low**：可选改进（如界面美化）

## 🎓 最佳实践

### 1. 角色命名
- ✅ 简洁易记（2-5 字）
- ✅ 体现功能（如「法务顾问」）
- ✅ 朗朗上口（避免生僻字）

### 2. 用途选择
- 🎯 **聚焦核心**：选择最匹配的用途
- 🎯 **勿贪多**：一个角色专注一个主要场景
- 🎯 **场景明确**：考虑实际使用环境

### 3. 风格匹配
- 💬 **符合场景**：正式场景用专业风格
- 💬 **考虑用户**：面向大众用友好风格
- 💬 **保持一致**：风格与用途协调

### 4. 专业领域
- 📚 **少而精**：1-3 个核心领域
- 📚 **相关性强**：领域之间有关联
- 📚 **注意合规**：特殊领域添加免责

### 5. 测试优化
- 🧪 **真实场景**：用实际问题测试
- 🧪 **多次迭代**：不满意就调整
- 🧪 **收集反馈**：记录用户评价

## ❓ 常见问题

### Q1: 提示词可以手动修改吗？
**A**: 可以！在第 3 步点击「查看生成的提示词」，可以预览和复制。创建后也可以在角色编辑页面进一步调整。

### Q2: 创建的角色可以修改吗？
**A**: 当然可以！所有创建的角色都支持随时编辑和更新配置。

### Q3: 智能推荐的准确性如何？
**A**: 推荐基于专家知识库和最佳实践，但具体情况还需结合实际需求判断。建议作为参考而非绝对标准。

### Q4: 支持保存草稿吗？
**A**: 支持！每一步都会自动保存到本地，关闭页面后回来可以继续。

### Q5: 可以导入导出角色配置吗？
**A**: 支持导出为 JSON 格式，方便分享和备份。导入功能正在开发中。

### Q6: 创建的提示词适用于哪些模型？
**A**: 生成的提示词兼容主流大语言模型（GPT-4、Claude、文心一言等）。系统会自动调整参数适配不同模型。
