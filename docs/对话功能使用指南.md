# RoleCraft AI - 对话功能使用指南

## 📖 快速开始

本指南介绍 RoleCraft AI 的对话增强功能，帮助您更高效地使用对话系统。

---

## 🎯 核心功能

### 1. 对话管理

#### 📝 重命名对话
让对话更易识别和管理。

**操作方法**:
1. 点击对话标题旁的 ✏️ 编辑图标
2. 输入新标题
3. 按 `Enter` 确认或点击 ✓ 按钮

**使用场景**:
- 为对话添加有意义的标题
- 根据对话内容分类命名

#### 📤 导出对话
保存重要对话内容到本地。

**操作方法**:
1. 点击右上角菜单按钮 (⋮)
2. 选择导出格式:
   - 📄 **导出为 Markdown**: 适合阅读和分享
   - 📋 **导出为 JSON**: 适合数据分析和备份

**快捷键**: 输入 `/export` 快速导出

**使用场景**:
- 保存重要咨询记录
- 备份有价值的对话
- 分享给团队成员

#### 📦 归档对话
整理不常用但需要保留的对话。

**操作方法**:
1. 点击右上角菜单按钮 (⋮)
2. 选择"归档"
3. 需要时再次点击"取消归档"

**快捷键**: 输入 `/archive` 快速归档

**使用场景**:
- 已完成的项目对话
- 暂时不需要的参考资料
- 保持对话列表整洁

#### 🔍 搜索对话
快速找到历史对话。

**操作方法**:
1. 点击搜索图标 🔍 或按 `Ctrl+K`
2. 输入搜索关键词
3. 按 `Enter` 执行搜索
4. 点击搜索结果打开对话

**快捷键**: 输入 `/search` 快速搜索

**搜索范围**:
- 对话标题
- 消息内容

**使用场景**:
- 查找特定主题的对话
- 回顾之前的讨论
- 定位重要信息

---

### 2. 消息交互

#### ✏️ 编辑消息
修正已发送的消息内容。

**操作方法**:
1. 鼠标悬停在您的消息上
2. 点击编辑按钮 ✏️
3. 修改内容
4. 点击"保存"或按 `Enter`

**使用场景**:
- 修正打字错误
- 补充遗漏信息
- 优化问题表述

#### 🔄 重新生成回复
获取不同的 AI 回答。

**操作方法**:
1. 鼠标悬停在 AI 回复上
2. 点击重新生成按钮 🔄
3. 等待 AI 生成新回复

**使用场景**:
- 对当前回答不满意
- 想要更多角度的解答
- 获取更详细的信息

#### 👍 点赞/点踩
反馈 AI 回答质量。

**操作方法**:
1. 鼠标悬停在 AI 回复上
2. 点击 👍（有用）或 👎（无用）
3. 按钮变色表示已评分

**使用场景**:
- 标记优质回答
- 反馈不准确的回答
- 帮助 AI 改进

#### 📋 复制回复
快速复制 AI 回答内容。

**操作方法**:
1. 鼠标悬停在 AI 回复上
2. 点击复制按钮 📋
3. 内容已复制到剪贴板

**使用场景**:
- 复制代码片段
- 保存重要信息
- 分享 AI 回答

#### 🗑️ 删除消息
移除不需要的消息。

**操作方法**:
1. 鼠标悬停在您的消息上
2. 点击删除按钮 🗑️
3. 确认删除

**使用场景**:
- 删除敏感信息
- 清理测试消息
- 移除错误内容

### 3. 快捷操作

#### ⌨️ 键盘快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Enter` | 发送消息 | 在输入框中按 Enter 发送 |
| `Ctrl+Enter` | 快速发送 | 强制发送（某些场景下使用） |
| `Ctrl+K` | 打开搜索 | 快速打开搜索框 |
| `Shift+Enter` | 换行 | 在输入框中换行 |
| `Esc` | 取消/关闭 | 关闭弹窗或取消操作 |

#### 🚀 快捷指令

在输入框中输入 `/` 开头的指令快速执行操作：

| 指令 | 功能 | 示例 |
|------|------|------|
| `/clear` | 清空对话 | 清除当前对话的所有消息 |
| `/help` | 显示帮助 | 打开快捷键和指令帮助 |
| `/export` | 导出对话 | 打开导出选项 |
| `/search` | 搜索对话 | 打开搜索框 |
| `/archive` | 归档对话 | 归档当前对话 |

**使用技巧**:
- 输入 `/` 后会显示可用指令列表
- 指令不区分大小写
- 可以输入部分指令后按 Tab 自动补全

## 💡 使用技巧

### 高效对话管理

1. **定期整理**: 每周归档已完成的对话，保持列表整洁
2. **命名规范**: 使用有意义的标题，如"项目 A-需求讨论-20260227"
3. **善用搜索**: 用关键词快速定位，避免手动翻找
4. **及时导出**: 重要对话及时导出备份

### 提升对话质量

1. **编辑优化**: 发送前仔细检查，发送后可编辑修正
2. **多次生成**: 不满意时尝试重新生成，获取更好答案
3. **积极反馈**: 使用点赞/点踩帮助 AI 学习
4. **善用复制**: 快速保存有价值的代码和文本

### 快捷键高手

1. **肌肉记忆**: 熟练使用 Ctrl+K 搜索，Ctrl+Enter 发送
2. **指令流**: 用 `/clear` 快速开始新话题
3. **Esc 救星**: 随时按 Esc 取消当前操作

## 🎨 界面说明

### 对话头部区域
```
┌─────────────────────────────────────────────┐
│ [AI] 助手名称 ✏️  [切换]  🟢 在线   [🔍] [⋮] │
```

### 消息区域
```
┌─────────────────────────────────────────────┐
│  [AI]  AI 回复内容...                        │
│       [📋] [🔄] [👍] [👎]                   │
│                                             │
│                          用户消息... [✏️] [🗑️]│
```

- **AI 消息**: 左侧显示，带操作按钮

```
┌─────────────────────────────────────────────┐
│ [📎] 输入消息... (/ 查看指令)          [➤]  │
```

- **附件**: 添加文件（待实现）

## 🔧 常见问题

### Q: 如何恢复已删除的消息？
A: 目前不支持消息恢复，删除前请确认。

### Q: 编辑消息会影响 AI 的后续回复吗？
A: 当前版本不会自动重新生成，需要手动点击重新生成。

### Q: 导出的文件保存在哪里？
A: 默认保存在浏览器的下载文件夹。

### Q: 归档的对话在哪里查看？
A: 需要在会话列表页面筛选"已归档"（待实现）。

### Q: 可以同时编辑多条消息吗？
A: 不支持，一次只能编辑一条消息。

### Q: 点赞/点踩后还能修改吗？
A: 可以，再次点击即可取消或修改评分。
