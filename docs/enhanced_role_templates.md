## 模板评分系统

### 评分维度

1. **用户满意度** (40%)
   - 使用后的评分反馈
   - 重复使用率
   - 推荐意愿

2. **使用效果** (30%)
   - 任务完成率
   - 对话质量评分
   - 问题解决率

3. **专业度** (20%)
   - 提示词质量
   - 知识准确性
   - 回答一致性

4. **创新性** (10%)
   - 功能独特性
   - 交互体验
   - 场景覆盖

### 评分计算公式

```
综合评分 = 用户满意度 × 0.4 + 使用效果 × 0.3 + 专业度 × 0.2 + 创新性 × 0.1
```

## 用户自定义模板

### 创建流程

1. **选择基础模板**
   - 从现有模板中选择最接近的作为起点

2. **修改系统提示词**
   - 调整角色定位
   - 添加专业技能
   - 定义行为准则

3. **配置能力参数**
   - 调整六维能力值
   - 设置回复风格
   - 定义交互模式

4. **测试验证**
   - 运行预设测试用例
   - 收集测试反馈
   - 迭代优化

5. **发布分享**
   - 设置公开/私有
   - 生成分享链接
   - 发布到模板市场

### 模板分享

- **分享链接**: 生成唯一 URL，可设置有效期和访问次数
- **一键导入**: 通过 JSON 配置快速导入
- **版本管理**: 支持模板版本追踪和回滚
