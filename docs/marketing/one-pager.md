### 5️⃣ API 集成，无缝连接

**开发者友好：**

```python
from rolecraft import Client

client = Client(api_key="your_key")

# 简单对话
response = client.chat.create(
    role_id="marketing_expert",
    message="写一份产品推广方案"
)
print(response.content)

# 流式输出
for chunk in client.chat.stream(
    role_id="tech_support",
    message="帮我 review 这段代码"
):
    print(chunk.content, end='')
```

**API 能力：**
- RESTful 接口
- 完整的文档
- SDK 支持（Python/Node.js）
- 用量统计和监控
