- 可开具增值税专用发票

### 用量计算

**API 调用计数：**
```
1 次对话 = 1 次 API 调用
无论消息长度和复杂度

示例：
- 用户发送 1 条消息 → 1 次调用
- AI 回复（任意长度）→ 已包含
- 重新生成 → +1 次调用
```

**文档存储：**
```
按文档数量计算，不按大小

示例：
- 专业版：100 个文档
- 每个文档最大 50MB
- 总存储空间：约 5GB
```

### 超额处理

**专业版超额：**
```
超出 10,000 次/天：
- 方案 1：降级到基础模型（GPT-3.5）
- 方案 2：购买额外包（¥50/1000 次）
- 方案 3：升级到企业版
```

**自动保护：**
- 用量达 80% 时邮件提醒
- 用量达 100% 时暂停服务
- 次日 0 点自动重置
