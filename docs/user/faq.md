## 1. 账户与登录

### Q1: 如何注册 RoleCraft AI 账号？

**A:** 
1. 访问 RoleCraft AI 官网
2. 点击右上角「注册」按钮
3. 填写邮箱、密码和昵称
4. 验证邮箱（查收确认邮件）
5. 完成注册

**注意：** 一个邮箱只能注册一个账号。

---

### Q2: 忘记密码怎么办？

**A:** 
1. 在登录页面点击「忘记密码」
2. 输入注册邮箱
3. 查收重置密码邮件
4. 点击邮件中的链接
5. 设置新密码

**提示：** 重置链接 24 小时后失效。

---

### Q3: 如何修改登录密码？

**A:** 
1. 登录后进入「设置」
2. 选择「安全设置」
3. 点击「修改密码」
4. 输入当前密码和新密码
5. 确认修改

**建议：** 定期更换密码，使用强密码。

---

### Q4: 支持第三方登录吗？

**A:** 
目前支持：
- ✅ 邮箱密码登录
- ⏳ Google 账号（即将支持）
- ⏳ GitHub 账号（即将支持）
- ⏳ 微信扫码（即将支持）

---

### Q5: 如何在多设备间同步数据？

**A:** 
RoleCraft AI 自动云端同步：
- 角色配置
- 对话历史
- 知识库文档
- 使用设置

**只需：** 使用同一账号登录即可。

## 2. 角色使用

### Q6: 如何创建自定义角色？

**A:** 
1. 点击「创建角色」按钮
2. 填写基础信息（名称、描述）
3. 配置系统提示词
4. 选择 AI 模型和参数
5. （可选）绑定技能和知识库
6. 点击「创建」完成

**详细指南：** [功能使用手册 - 角色编辑器](./user-guide.md#23-角色编辑器)

---

### Q7: 内置角色模板可以修改吗？

**A:** 
内置模板本身不可修改，但你可以：
1. 使用模板创建角色
2. 基于模板克隆
3. 自定义修改克隆的角色

**步骤：**
- 在角色市场找到喜欢的模板
- 点击「使用」或「克隆」
- 在「我的角色」中编辑

---

### Q8: 一个账号可以创建多少个角色？

**A:** 

| 订阅计划 | 角色数量 |
|---------|---------|
| 免费版 | 10 个 |
| 专业版 | 50 个 |
| 企业版 | 无限制 |

**注意：** 删除角色可释放名额。

---

### Q9: 如何分享我的角色？

**A:** 
**分享给团队成员：**
1. 进入角色编辑页
2. 点击「分享」
3. 选择团队成员
4. 设置权限（只读/可编辑）

**公开分享：**
1. 设置为「公开」
2. 生成分享链接
3. 发送链接给朋友

**注意：** 公开角色所有用户可见。

---

### Q10: 角色可以导出备份吗？

**A:** 
是的，支持导出：
1. 进入「我的角色」
2. 点击角色「⋮」菜单
3. 选择「导出」
4. 下载 JSON 配置文件

**导入恢复：**
1. 点击「导入角色」
2. 上传 JSON 文件
3. 确认导入

## 3. 对话功能

### Q11: 快速问答和任务模式有什么区别？

**A:** 

| 特性 | 快速问答 | 任务模式 |
|------|---------|---------|
| 上下文 | 不保留 | 保留 20 条 |
| 响应速度 | 更快 | 标准 |
| 适用场景 | 简单问题 | 复杂任务 |
| Token 消耗 | 较少 | 较多 |

**建议：** 根据场景选择合适的模式。

---

### Q12: 对话历史会保存多久？

**提示：** 重要对话建议手动保存或导出。

## 4. 知识库

### Q16: 知识库如何处理上传的文档？

**A:** 
**处理流程：**
1. **上传** - 用户提交文档
2. **解析** - 提取文本内容
3. **分块** - 按 500 字/块分割
4. **向量化** - 转换为向量
5. **索引** - 存入向量数据库
6. **可用** - 可被检索使用

**处理时间：** 通常 1-5 分钟，取决于文档大小。

---

### Q17: 一个角色可以关联多少个文档？

**A:** 

| 订阅计划 | 文档数量 |
|---------|---------|
| 免费版 | 20 个 |
| 专业版 | 100 个 |
| 企业版 | 无限制 |

**建议：** 关联相关文档，避免过多影响检索质量。

---

### Q18: 文档更新后需要重新处理吗？

**A:** 
是的，文档更新流程：
1. 删除旧文档
2. 上传新文档
3. 自动重新处理
4. 角色自动使用新版本

**未来优化：** 支持文档版本管理和增量更新。

---

### Q19: 知识库检索准确度如何？

**A:** 
**影响因素：**
- 文档质量（清晰度、结构）
- 分块大小（默认 500 字）
- 检索配置（Top-K、阈值）
- 问题表述（具体明确）

**优化建议：**
- 上传前清理文档格式
- 添加清晰的标题和段落
- 使用关键词检索辅助
- 调整相似度阈值

**典型准确度：** 85-95%（取决于场景）

## 5. API 集成

### Q21: 如何获取 API 密钥？

**A:** 
1. 登录 RoleCraft AI
2. 进入「API 平台」
3. 点击「创建密钥」
4. 输入密钥名称
5. 选择权限范围
6. 点击创建
7. **立即复制密钥**（只显示一次）

**安全提示：** 妥善保管密钥，不要泄露。

---

### Q22: API 调用有限制吗？

**A:** 

| 订阅计划 | 调用限制 |
|---------|---------|
| 免费版 | 100 次/天 |
| 专业版 | 10,000 次/天 |
| 企业版 | 定制 |

**超出限制：** 返回 429 错误，需升级或等待重置。

---

### Q23: API 响应格式是什么？

**A:** 
**请求格式：** JSON
**响应格式：** JSON

**示例响应：**
```json
{
  "success": true,
  "data": {
    "message_id": "msg_123",
    "content": "你好！有什么可以帮你的？",
    "role_id": "role_456",
    "created_at": "2026-02-27T10:00:00Z"
  },
  "usage": {
    "tokens": 150,
    "cost": 0.002
  }
}
```

---

### Q24: 支持流式输出吗？

**A:** 
是的，支持 SSE（Server-Sent Events）流式输出。

**示例：**
```bash
curl -X POST https://api.rolecraft.ai/v1/chat/stream \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"role_id": "role_123", "message": "你好"}'
```

**优势：** 实时显示，用户体验更好。

---

### Q25: API 调用失败如何处理？

**A:** 
**常见错误码：**

| 错误码 | 说明 | 解决方案 |
|-------|------|---------|
| 400 | 请求参数错误 | 检查请求格式 |
| 401 | 认证失败 | 检查 API 密钥 |
| 403 | 权限不足 | 检查密钥权限 |
| 404 | 资源不存在 | 检查 ID 是否正确 |
| 429 | 超出限额 | 等待或升级 |
| 500 | 服务器错误 | 联系技术支持 |

**重试建议：** 对 5xx 错误实施指数退避重试。
