# RoleCraft AI - 快速开始指南

> 5 分钟快速上手 RoleCraft AI

---

## 🎯 本指南适合谁

- 第一次使用 RoleCraft AI 的新用户
- 想快速了解核心功能的用户
- 需要立即开始创建 AI 角色的用户

---

## 📋 前提条件

- 已注册 RoleCraft AI 账号
- 现代浏览器（Chrome、Edge、Safari）
- 网络连接正常

---

## 🚀 5 分钟快速上手

### 第 1 分钟：登录与首页

1. 访问 RoleCraft AI 网站
2. 使用邮箱和密码登录
3. 进入首页仪表板

**首页你将看到：**
- 📊 使用统计（角色数、对话数）
- 🎭 最近使用的角色
- 💬 最近的对话列表
- ⭐ 推荐角色模板

---

### 第 2 分钟：选择你的第一个角色

RoleCraft AI 内置了 8 个专业角色模板：

| 角色 | 用途 | 适合场景 |
|------|------|----------|
| 🤖 智能助理 | 全能型办公助手 | 日常办公、邮件撰写、日程管理 |
| 📈 营销专家 | 营销策划与内容创作 | 文案写作、活动策划、市场分析 |
| ⚖️ 法务顾问 | 合同审查与法律咨询 | 合同审核、法规查询、风险评估 |
| 💰 财务助手 | 财务报表与税务咨询 | 报表分析、税务规划、预算编制 |
| 💻 技术支持 | IT 问题诊断与解决 | 故障排查、代码审查、架构建议 |
| 🏢 前台接待 | 客户接待与预约管理 | 访客登记、预约安排、电话接听 |
| 👥 HR 专员 | 招聘与员工关系 | 简历筛选、面试安排、政策解答 |
| 📦 产品经理 | 需求分析与产品设计 | 用户调研、PRD 撰写、竞品分析 |

**操作：**
1. 点击左侧导航栏的「角色」
2. 浏览角色市场
3. 点击任意角色卡片上的「使用」按钮

---

### 第 3 分钟：开始第一次对话

进入角色对话界面后：

1. **查看欢迎语** - 角色会自我介绍并说明能力
2. **输入你的问题** - 在底部输入框输入问题
3. **发送消息** - 按 Enter 或点击发送按钮
4. **等待回复** - AI 会实时生成回复（流式输出）

**示例对话：**

```
你：你好，我需要写一份产品推广方案

营销专家：你好！很高兴帮你制定产品推广方案。

为了给你更精准的建议，请告诉我：
1. 你的产品是什么？
2. 目标用户群体是谁？
3. 推广预算大概多少？
4. 主要推广渠道偏好？

有了这些信息，我可以为你制定一份完整的推广方案！
```

---

### 第 4 分钟：尝试高级功能

#### 📎 上传文档（知识库）

让 AI 基于你的文档回答问题：

1. 点击输入框旁的「📎」按钮
2. 选择 PDF、Word 或 TXT 文件
3. 等待文档处理完成
4. AI 会基于文档内容回答

#### ⚡ 使用快捷命令

在输入框输入 `/` 触发快捷命令：

- `/总结` - 总结长文档或对话
- `/翻译` - 翻译内容为指定语言
- `/分析` - 深度分析数据或内容
- `/改写` - 改写文案或文章
- `/扩展` - 扩展简短内容为详细版本

**示例：**
```
/总结 请总结这份会议记录的关键要点
```

#### 💾 保存对话

1. 点击右上角「💾 保存」按钮
2. 输入对话标题
3. 点击保存
4. 稍后可在「历史记录」中继续

---

### 第 5 分钟：创建自定义角色

打造专属你的 AI 数字员工：

1. 点击「创建角色」按钮
2. 填写基础信息：
   - 角色名称（如：我的写作助手）
   - 角色描述（一句话介绍）
   - 选择头像
3. 配置系统提示词：
   ```
   你是一位专业的写作助手，擅长帮助用户
   撰写、编辑和优化各类文章。你的风格
   亲切友好，善于给出具体建议。
   ```
4. 选择模型和参数：
   - 模型：选择 AI 模型
   - 温度：控制创造性（0-1）
5. 点击「创建」完成
