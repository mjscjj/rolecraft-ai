```

```

### 3. 忘记密码流程 ✅ NEW
```
登录页 → 忘记密码 → 输入邮箱 → 发送重置链接 → 输入令牌 → 重置密码
```

### 4. 创建 AI 角色流程 ✅
```
仪表盘 → 创建角色 → 选择模板/自定义 → 配置提示词 → 配置知识库 → 完成
```

### 5. AI 对话流程 ✅
```
选择角色 → 进入对话 → 发送消息 → AI 回复 → 消息操作 → 导出对话
```

### 6. 知识库管理流程 ✅
```
知识库页面 → 上传文档 → 文档列表 → 搜索文档 → 文件夹管理 → 批量操作
```

### 7. 数据分析流程 ✅
```
数据分析页面 → 查看仪表盘 → 查看统计 → 生成报告 → 导出报告
```
