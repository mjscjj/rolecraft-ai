# RoleCraft AI - UI 设计规范

## 1. 设计原则

### 1.1 核心设计理念
- **清晰优先**：信息层级分明，降低认知负担
- **效率导向**：减少操作步骤，提供快捷入口
- **专业可信**：稳重的视觉语言，建立用户信任
- **友好亲近**：适当的圆角和动效，降低 AI 的距离感

### 1.2 设计关键词
智能 · 专业 · 简洁 · 高效 · 可信赖

## 4. 间距系统

基础单位为 4px，所有间距遵循 4 的倍数。

| Token | 值 | 用途 |
|-------|-----|------|
| space-1 | 4px | 图标内边距、小间隙 |
| space-2 | 8px | 紧凑间距、行内元素 |
| space-3 | 12px | 表单元素间距 |
| space-4 | 16px | 标准间距、卡片内边距 |
| space-6 | 24px | 区块间距 |
| space-8 | 32px | 大区块间距 |
| space-12 | 48px | 页面级间距 |

### 4.2 布局规范

**页面边距：**
- Desktop: 32px
- Tablet: 24px
- Mobile: 16px

**容器最大宽度：**
- 全宽: 100%
- 标准: 1200px
- 窄版: 800px
- 表单: 600px

## 7. 组件规范

### 7.1 按钮 (Button)

**主要按钮 (Primary):**
- 背景：渐变（#00d4aa → #0891b2）
- 文字：白色，14px，500 字重
- 内边距：12px 24px
- 圆角：8px
- 悬停：亮度增加，轻微上浮
- 禁用：透明度 50%

**次要按钮 (Secondary):**
- 背景：透明
- 边框：1px solid #e2e8f0
- 文字：#334155
- 悬停：背景 #f8fafc

**幽灵按钮 (Ghost):**
- 背景：透明
- 文字：#64748b
- 悬停：背景 rgba(0,212,170,0.1)

### 7.2 输入框 (Input)

**文本输入框：**
- 高度：44px
- 内边距：12px 16px
- 边框：1px solid #e2e8f0
- 圆角：8px
- 聚焦：边框色 #00d4aa，外发光
- 占位符：#94a3b8

**文本域：**
- 最小高度：100px
- 可拖拽调整大小
- 其余同输入框

### 7.3 卡片 (Card)

**标准卡片：**
- 背景：白色
- 圆角：12px
- 阴影：shadow-md
- 内边距：24px
- 悬停：shadow-lg，轻微上浮

**角色卡片：**
- 尺寸：280px × 200px
- 包含：头像、名称、描述、标签
- 操作按钮在悬停时显示

### 7.4 标签 (Tag)

**分类标签：**
- 背景：rgba(0,212,170,0.1)
- 文字：#0891b2
- 内边距：4px 12px
- 圆角：9999px
- 字号：12px

**状态标签：**
- 成功：绿底绿字
- 处理中：蓝底蓝字
- 失败：红底红字

### 7.5 导航 (Navigation)

**侧边导航：**
- 宽度：260px
- 背景：白色
- 项高：44px
- 选中：左侧 3px 强调色边框
- 图标：20px

**顶部导航：**
- 高度：64px
- 背景：白色 + 底部阴影
- Logo 区 + 搜索 + 用户区

**第三步：技能配置**
- 技能库列表（可拖拽）
- 已选技能区域
- 技能参数配置面板

**第四步：知识绑定**
- 文档选择器（多选）
- 检索参数设置
- 已绑定文档列表

**第五步：测试发布**
- 实时预览对话窗口
- 发布按钮

## 9. 动效规范

### 9.1 过渡动画
- 标准时长：200ms
- 缓动函数：cubic-bezier(0.4, 0, 0.2, 1)
- 页面切换：300ms

### 9.2 微交互
- 按钮悬停：scale(1.02)，150ms
- 卡片悬停：translateY(-4px)，shadow-lg，200ms
- 加载动画：旋转圆圈，1s 线性无限
- 成功提示：从上方滑入，300ms

### 9.3 打字机效果
AI 回复时文字逐个显示，延迟 15ms/字

## 10. 响应式适配

### 10.1 断点
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: 1024px - 1440px
- Large: > 1440px

### 10.2 移动端适配
- 侧边栏变为底部导航
- 卡片单列展示
- 表格变为卡片列表
- 输入框全宽
