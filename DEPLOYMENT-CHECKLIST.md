## 🎯 上线步骤

### 1. 生产环境部署
```bash
# 1. 克隆项目
git clone https://github.com/mjscjj/rolecraft-ai.git
cd rolecraft-ai

# 2. 后端部署
cd backend
cp .env.example .env
# 编辑 .env 配置数据库和 API Key
go build -o server cmd/server/main.go
./server

# 3. 前端部署
cd ../frontend
npm install
npm run build
# 部署 dist 到 Nginx/Vercel

# 4. 数据库初始化
# SQLite: 自动创建
# PostgreSQL: 运行迁移脚本
```

### 2. 环境变量配置
```bash
# 后端 .env
DATABASE_URL=postgresql://user:pass@localhost:5432/rolecraft
OPENROUTER_API_KEY=sk-or-v1-xxx
ANYTHINGLLM_API_KEY=xxx
JWT_SECRET=your-secret-key
PORT=8080

# 前端 .env
VITE_API_BASE_URL=https://api.rolecraft.ai
VITE_OPENROUTER_KEY=sk-or-v1-xxx
```
